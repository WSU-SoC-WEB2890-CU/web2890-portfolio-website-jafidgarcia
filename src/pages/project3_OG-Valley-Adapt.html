03 - Ogden Valley Adaptive

See Prototype

Client: Ogden Valley Adaptive

Technologies: Photoshop, Adobe XD, Material Design

Project Description: For this project I was tasked with redisgning a website into a mobile application that the non-profit Ogden Valley Adaptive can use to make their non-profit more accessible to it's target audience. This was a Material Design App for Android. We started with paper prototypes and tested the main functionalities with real users. We then did the same for the medium and high fidelity prototypes making changes along the way.

Design: This was an evolving design as we tested with users to see what design elements would work best for easy navigation and usability. The app design followed Material Design rules in order to design for an Android app.

Development: The developement included paper prototyping as well as using Adobe XD to create the medium and high fidelity prototypes.
