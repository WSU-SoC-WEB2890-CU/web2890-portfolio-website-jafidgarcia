02 - Adventure Sports Media

See Live Site

Client: Web 2500 Final Project

Technologies: HTML, CSS, SCSS, JavaScript Bootstrap, Git, Netlify, Webpack, Photoshop, XD

Project Description: Adventure Sports Media needed a website to showcase their abilities and services to potential clients and companies. Because of their niche clientele, we conducted extensive User Research to ensure we designed and developed an optimized website for them.

Design: We began with a competitor analysis to understand the current market for action sports photography websites. We then conducted surveys to create user Persona’s to identify the needs of potential users. Once we understood which elements would work best for this website, we created Wireframes and conducted Navigation Testing to ensure a frictionless user experience.

Development: In order to code to the client design we used a combination of customized SASS and Bootstrap to create grid layout elements and functionality for the website throughout its responsive display sizes. One of the challenges for this project was customizing Bootstrap classes to be different for Mobile displays vs Desktop displays by changing the parameters of the hamburger menu collapse class to cover the entire screen size rather than just the standard small dropdown menu card that Bootstrap provides.
